V008 Windows Claude Code 安装流程
code.jimuxyz.com

Windows 安装 Claude Code 到第一次提问

STEP 1

文档

打开 /docs/cli，找到 Windows 区域。

/docs/cli
STEP 2

Key

确认用 Claude Code Key，不混用。

Claude Key
STEP 3

环境

PowerShell 先看 Node、npm、Git。

node npm git
STEP 4

安装

安装入口需录屏前实测确认。

claude-code
STEP 5

配置

写入 API 地址和 Key，完整 Key 不上屏。

sk-***
STEP 6

验证

重开 PowerShell 后看 version、doctor、auth。

claude doctor
STEP 7

日志

干净目录第一次提问，再回 /logs 核对记录。

/logs success

正常判断

版本能输出，doctor 能跑，第一次提问能返回，日志能对上。

常见错法

Key 类型混用、配置完不重开终端、把环境缺失当成 Key 错。

总结文档 → Key → 环境 → 安装 → 配置 → 验证 → 第一次提问 → 日志核对。
