安装到第一次提问：按这条链路核对

只保留本条视频已有关键流程；服务器、账号、日志均使用脱敏示意。
STEP 1

ECS / SSH 入口

确认服务器可连，公网 IP 和实例信息录屏需遮挡。

STEP 2

普通用户与权限

看 whoami、id、pwd，避免安装用户和运行用户错位。

STEP 3

Git / Node / curl

先查基础工具，再判断是否能执行安装脚本。

STEP 4

安装 Claude Code

使用本地资料确认的官方安装命令。

STEP 5

认证与配置

登录状态、第三方 Key/API 入口需录屏前实测确认。

STEP 6

第一次提问

进入干净目录，只问只读问题，不碰真实项目。

STEP 7

日志 / 状态核对

按终端输出、配置和日志页拆开排查。

install and check commands
# 本条确认使用的安装命令
$ curl -fsSL https://claude.ai/install.sh | bash
$ claude --version
$ claude doctor
$ claude auth status --text

排查标签

终端输出 配置文件 Key/API 地址需实测 权益状态 日志页需实测 网络 权限 项目目录
总结 按 SSH、用户、环境、安装、认证、项目目录和日志状态逐段核对。
