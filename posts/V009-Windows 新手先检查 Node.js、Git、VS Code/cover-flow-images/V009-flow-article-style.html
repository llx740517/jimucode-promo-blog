V009 Windows 安装前环境检查
code.jimuxyz.com

Windows 新手先检查 Node.js、Git、VS Code

STEP 1

PowerShell

先看提示符里有没有 PS，确认当前终端。

PS C:\...
STEP 2

Node/npm

看版本和 where.exe 路径是否明确。

node -v
STEP 3

Git

确认 Git for Windows 命令能跑。

git --version
STEP 4

VS Code

确认 code 命令能被终端找到。

code --version
STEP 5

目录

准备空目录，不直接进真实项目。

code .
STEP 6

文档

环境通过后，再去复制安装命令。

/docs/cli

通过标准

Node/npm、Git、code 命令都能输出，code . 能打开干净目录。

常见错法

PATH 未刷新、软件未装完整、路径指向旧目录。

总结PowerShell → Node/npm → Git → VS Code → 干净目录 → CLI 文档。
