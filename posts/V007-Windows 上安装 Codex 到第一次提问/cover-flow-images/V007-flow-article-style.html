Windows 安装 Codex 到第一次提问

STEP 1

环境

PowerShell 先看 Node.js 和 npm，版本不够先处理环境。

node npm
STEP 2

检查

执行页面环境检查脚本，确认机器满足前置条件。

install check
STEP 3

安装

只复制 Codex 的 Windows PowerShell 安装命令。

npm install
STEP 4

配置

用 Codex Key 写入 .codex 配置，完整 Key 不进画面。

sk-***
STEP 5

提问

新终端验证后，在干净目录完成第一次提问。

codex exec
STEP 6

日志

回到 /logs 看当天请求记录是否能对上。

/logs success

正常判断

版本能输出、doctor 能跑、第一次提问能返回，日志能对上。

常见错法

不查 Node/npm、复制错命令、配置完不重开终端、直接进真实项目。

总结环境 → 检查 → 安装 → 配置 → 第一次提问 → 日志核对；失败就停在当前层排查。
