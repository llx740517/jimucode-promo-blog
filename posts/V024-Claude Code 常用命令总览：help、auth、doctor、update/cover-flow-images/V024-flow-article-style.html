Claude Code CLI 入门命令顺序：先查 → 再修

保留原流程：终端 → version/help → auth → doctor → update/install → 进入项目
STEP 1

打开终端

进入干净目录，不展示真实项目。

STEP 2

版本 / help

`claude --version` 和 `claude --help` 先确认入口。

STEP 3

auth

登录、状态、退出；账号信息必须打码。

STEP 4

doctor

检查安装和配置，路径与日志需预览。

STEP 5

update/install

区分安装方式，不把更新当万能修复。

STEP 6

进入项目

项目读取、恢复会话、模型选择放后续。

底部检查顺序

终端输出版本登录状态配置文件网络权限安装方式项目目录

安全提示

`setup-token` 只展示 help 或脱敏示意，不展示真实 Token；auth 和 doctor 输出需录屏前实测确认。

记忆线：version/help → auth → doctor → update/install → 再进入项目或继续会话
