V017 浅色图文流程

SSH 登录阿里云服务器三种入口

保留本条关键流程：ECS / 安全组 → 登录三要素 → 三端入口 → 验证 → 排查。
STEP 1

ECS 实例准备

实例运行中，Linux 系统，有公网 IP。控制台入口需录屏前实测确认。

STEP 2

安全组检查

入方向 TCP 22，来源建议限制当前可信 IP，不长期全网开放。

STEP 3

登录三要素

公网 IP、用户名、密码或密钥；默认用户以实例设置为准。

STEP 4

三种本地入口

Windows PowerShell、macOS Terminal、Linux Terminal；Workbench 辅助。

STEP 5

登录后验证

运行 whoami、hostname、pwd、uname -a；确认后 exit。

WinPowerShell 先跑 ssh -V。
Macchmod 400 后再 ssh -i。
Linux路径不同，结构相同。
WebWorkbench 仅作辅助验证。
排障顺序

终端输出 / 安全组 22 / 公网 IP / 用户名 / 密钥权限 / 实例状态 / 网络 / 项目目录
