V006 Claude Code 登录流程
code.jimuxyz.com

Claude Code 新手从 0 到 1：先登录再进入项目

STEP 1

产品线

打开 /pricing，确认当前配置的是 Claude Code。

/pricing
STEP 2

Key

在 /keys 创建测试 Key，完整内容不进公开画面。

Key: sk-***
STEP 3

环境

PowerShell 先看 Node、npm、Git 和 Claude 命令。

node npm git
STEP 4

鉴权

重开终端后跑 auth status 和 doctor。

auth status
STEP 5

项目

进入干净目录，先让 Claude Code 读取项目。

claude -p
STEP 6

日志

回到 /logs 看当天请求记录是否能对上。

/logs success

正常判断

auth / doctor 有可解释结果，干净项目能读取，日志能对上。

常见错法

混用 Key、旧终端不重开、直接进入真实项目，都会放大排查难度。

总结产品线 → Key → 环境变量 → 鉴权状态 → 项目目录 → 日志核对；失败就停在当前层排查。
