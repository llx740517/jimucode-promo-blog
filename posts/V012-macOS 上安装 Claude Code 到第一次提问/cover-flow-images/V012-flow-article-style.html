macOS 安装 Claude Code 到第一次提问

STEP 1
检查环境

Terminal / zsh

Node.js 18+ 与 npm

node --version command -v claude
STEP 2
安装本体

以官方安装脚本为主。

Homebrew 是备选。

curl -fsSL https://claude.ai/install.sh | bash
STEP 3
创建 Key

进入积木 /keys。

产品线选 Claude Code。

Key：你的Claude_API_KEY 全程打码
STEP 4
写入配置

macOS zsh 通常写到 ~/.zshrc。

ANTHROPIC_BASE_URL=https://code.jimuxyz.com/p source ~/.zshrc
STEP 5
首次运行

先进入可删除测试目录。

问题只读，不直接改文件。

mkdir -p ~/Desktop/claude-code-demo claude
STEP 6
日志核对

查看 /logs 的时间、产品线、状态。

明细需登录后确认。

status: success model: example

正常

先让 claude 命令可用，再配置积木 Key 与 API 地址。

常见错法

把 Codex Key、旧安装方式和未生效配置混在一起查。

为什么这样做

能把安装、配置、权限、日志四类问题拆开定位。

终端输出zsh 配置Claude Code KeyAPI 地址日志页网络代理项目目录
检查环境 → 安装 → 配置 → 只读验证 → 看日志
